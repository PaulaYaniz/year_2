```
START
INPUT ARRAY sales
big = 0
small = 0

FOR i=0 to LEN(sales):
  IF sales[i] > sales[i+1]:
    temp = sales[i]
    sales[i] = sales[i+1]
    sales[i+1] = temp
  END IF
END FOR

big = sales[0]
small = sales[LEN(sales)-1]
OUT = big-small
END
```
